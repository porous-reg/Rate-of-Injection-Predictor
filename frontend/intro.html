ROI waveform predictor

Predict the full ROI waveform from pressure and duration.

This first release uses a fixed experimental context: injector 800 at 30 C. The user enters rail pressure and injection duration, and the runtime returns the full ROI waveform with summary metrics.

What this site is

A deployment-friendly ROI webapp with the same interaction pattern as the plume site: top tabs, a left control rail, and a right workspace for plots and tables.

What this site is not

It is not a geometry-driven injector tool in this release. Injector hole count, hole pattern, and other geometry-aware fields are reserved for later versions.

Future extension

The site shell is intentionally kept reusable so later ROI versions can add more operating conditions or geometry-aware inputs without changing the overall layout.
