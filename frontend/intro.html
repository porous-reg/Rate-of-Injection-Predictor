ROI waveform predictor

Predict the full ROI waveform from operating conditions.

This site wraps the ROI runtime bundle into a simple web workflow: choose an injector, set the condition, and inspect the predicted waveform with summary metrics.

What this site is

A deployment-friendly ROI webapp with the same structural interaction pattern as the plume site: top tabs, a left control rail, and a right workspace for plots and tables.

What this site is not

It is not a plume-physics application and it does not use geometry-driven collapse logic. The first release is condition-only.

Future extension

The interface is designed so geometry-aware inputs such as injector hole count can be added later without changing the overall site structure.
